Inventory Management System (TypeScript / HTML / CSS)

Project Overview
This is a web-based inventory management system developed with **TypeScript**, **HTML**, and **CSS**.
It allows users to add, update, search, delete, and view inventory items with full validation and interactive UI.

File Structure
- index.html – Main interface
- app.ts – Core TypeScript logic
- styles.css – Page styling
- README.md – Project documentation

How to Run
Compile TypeScript to JavaScript:
Open index.html in any browser
Use the form to manage inventory items

Technologies Used
- TypeScript
- HTML5
- CSS3
- Vanilla JavaScript (compiled from TS)

Function Description
- Add/Update Item: Fill all required fields and save
- Search: Type in the search box to filter items
- Show All: Display all items in inventory
- Show Popular: Display items marked as popular
- Delete: Remove items by name or direct delete button
